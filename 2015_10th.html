ゲーム業界における開発スタイルの歴史

講師: 武岡 幸夫

家庭用ゲーム機の黎明期から現在に至るまでの開発スタイルの歴史について語ります。
８ビットCPUで表現もプアだった時代にどのようにゲームを開発していたか、それがどのように変遷し、現在のような壮大なゲームを 多くのスタッフで分担して開発していけるようになったかを知ることで、組み込み開発における小規模プロジェクト・大規模プロジェクトでの開発スタイルの参考になるのではないかと思います

Java Puzzle Attack !!

講師: 前田 寛之

研修時間内に出す問題を実際にその場でプログラム (Java) を作成して問題を解いていきます。

*基本 Java のみの解説サポートになります。
